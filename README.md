**Tea Shop Website**

A modern and responsive Tea Shop website built using HTML, CSS, and Tailwind CSS.
This project provides an online presence for a tea shop where users can explore tea products, monthly packages, reviews, job opportunities, and manage their accounts.

**Project Overview**

The Tea Shop Website is designed to offer a smooth and user-friendly experience for customers who love tea.
It includes multiple pages such as Home, Order Packages, Login/Signup, Reviews, and Careers.

The website is fully responsive and styled using Tailwind CSS for a clean and modern user interface.

**Technologies Used**

HTML5 – Page structure

CSS3 – Custom styling

Tailwind CSS – Responsive and utility-first styling

JavaScript (optional) – For interactivity such as form validation
